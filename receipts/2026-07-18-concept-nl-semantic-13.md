# 10,000 WordNet-linked concepts across thirteen language lenses

## Source and license

This lane pins the Open Multilingual Wordnet data repository at commit
`406bf83b3c507a3d1f26e88252d5d66893fd36bf`. OMW's tab format links lemmas to
Princeton WordNet 3.0 `offset-pos` synsets. The selected source class is OMW's
automatically extracted Wiktionary tables for all thirteen languages, rather
than a confidence-obscuring mixture of curated and automatic wordnets.

The Wiktionary source text is dual-licensed under CC BY-SA 3.0 and GFDL 1.1 or
later. The upstream license text is retained verbatim in
`cognition/concept-nl-semantic-13-omw-license.txt`. Per-file SHA-256 values,
upstream commit, selection rule, and selected-table hash live in
`cognition/concept-nl-semantic-13-source-manifest.txt` and the Form-native
metadata cell. OMW and the original Wiktionary contributors remain the source;
this repo is only a compact derived carrier.

## Deterministic selection

The anchor set is the first 10,000 unique English lemma-bearing PWN 3.0 synsets
in upstream source order. For every selected synset and language, the pinned
surface is the first non-empty lemma in that language's source file. Empty
upstream records are skipped; this matters because sixteen English synsets
begin with an empty lemma record before a real lemma.

The result is a 10,001-line TSV (one header plus 10,000 concepts), SHA-256
`b063c69f3a0f80a272a890fa5e5e6d43bca50105353114b5b69e6f580a2dcc30`.
No rented or seed translation is used.

## Form-native surface

The 677 KB TSV is paired with 10,001 generated byte offsets. Form uses that
index to access a concept row, generate a real mapped label, expose absence and
review state, detect the first exact concept for a label, and recompute each
locale's coverage over all 10,000 rows. First-match detection is explicit:
WordNet lemmas can be polysemous, so this is not word-sense disambiguation.

`cognition/concept-nl-semantic-13-runtime.fk` is the operational carrier, not
a test fixture: it opens the committed TSV/index once and exposes live lookup,
generation, detection, and coverage functions over that reusable runtime.

Index 440 (`00292937-a`) is a live all-thirteen witness: `busy`, `sibuk`,
`ocupado`, `occupé`, `ocupado`, `bizi`, `beschäftigt`, `занятой`, `忙`,
`忙しい`, `مشغول`, `व्यस्त`, `meşgul`.

`cognition/tests/concept-nl-semantic-13-live.fk` prints those UTF-8 values from
the loaded TSV through the Form APIs, one row per lens. It also prints the
mapped/absent counts and the exact-detection result, plus a real absent record:
Indonesian at `00002098-a` is empty and reports `absent`. This is executable
data, not a prose-only fixture.

The pure metadata/API contract is suitable for four-way proof. The complete
UTF-8 TSV witness is fkwu-native: proof walkers do not carry `read_file`, and
the minimal TypeScript walker explicitly lacks byte-identical non-Latin
substring reconstruction. That seam is named rather than bypassed.

## Remaining floor

- No label is reviewed or canonicalized.
- Only the first upstream lemma is retained; synonyms remain in the source,
  outside this compact projection.
- Detection returns the first exact match and does not disambiguate senses.
- These are lexical labels, not sentence generation or translation fluency.
- PWN relations are not yet carried; this lane shares concept identity but not
  the full hypernym/meronym graph.
