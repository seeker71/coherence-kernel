# Thirteen bounded natural-language lenses

This receipt does **not** claim thirteen-language fluency. It records one
executable Form-native phrase-semantic lens for each of exactly thirteen
registry seats: `en`, `id`, `es`, `fr`, `pt-br`, `sw`, `de`, `ru`, `zh`, `ja`,
`ar`, `hi`, and `tr`.

"Top thirteen" here has one narrow, executable selection criterion: these are
the first thirteen natural-language rows in `form/form-stdlib/locale-registry.fk`
whose generator/proof is not `pending`, in that registry's order as observed on
2026-07-18. `nl13-selection-evidence` carries that source, date, ordered code
list, and proxy status in Form. This is a repository-readiness ranking, **not**
a ranking by world population, number of speakers, importance, or quality;
population-ranking parity is explicitly `0`. Sanskrit is the next non-pending
registry row and falls outside only because this organ is exactly thirteen.

## Audit floor

- `form/form-stdlib/locale-registry.fk` names locale lanes, but its working
  renderer deliberately returns the source unchanged for most of them.
- `form/form-stdlib/nl-translate.fk` is a real neutral-pivot path, bounded to a
  small English/Indonesian property grammar.
- `cognition/speak-locale.fk` composes three locale templates.
- `form/form-stdlib/text-tokenize.fk` explicitly documents its ASCII boundary;
  it drops CJK and breaks other UTF-8 words.
- `cognition/nl-emitter.fk` proves a three-token greedy/EOS shape. The larger
  autoregressive generators are not thirteen-locale language models.

## What landed

`cognition/nl-lenses-13-data.fk` pins exactly thirteen phrase packs. Nine
presence phrases cite existing text/audio receipts in the body. Swahili,
Russian, Hindi-script, and Turkish currently carry the explicit provenance
`seed-pinned-unreviewed`; they are work awaiting native-speaker review, not
authority smuggled in as proof.

`cognition/nl-lenses-13.fk` supplies:

1. exact-exemplar language detection over all thirteen rows (`str_eq`, because
   the minimal TypeScript proof walker explicitly does not provide byte-identical
   Unicode substring reconstruction);
2. generation for four neutral meanings (presence, enquiry, affirmation, and
   a parameterized grounded frame);
3. bounded cross-lens transformation through a numeric meaning, including
   preservation of caller-supplied detail through the structured `nl13-reframe`
   operation. Raw text-to-text frame parsing remains bounded to the live `Form`
   exemplar; it is not presented as general Unicode segmentation.

The detector is not statistical language identification. The transformer is
not general translation. Unknown text and unknown target codes fail closed.

## Witnesses

`cognition/tests/nl-lenses-13-band.fk` gives one independent bit to each lens.
Each example detects the presence/enquiry/affirmation/frame phrases, generates
the parameterized frame, transforms presence into the next lens, and preserves
`Form` while transforming the frame. Full verdict: `8191`.

Observed on 2026-07-18 from the same four-file recipe:

```text
fkwu                 8191
Go proof walker      8191
Rust proof walker    8191
TypeScript walker    8191
git diff --check     pass
```

The TypeScript crossing exposed and corrected a real boundary. Its minimal
proof walker documents a Latin-1-only `str_byte_at`, so byte-composed
`str_find` cannot reconstruct source literals in Cyrillic, Han, kana, Arabic,
or Devanagari. Fixed exemplars now use `str_eq`, which is Unicode-correct on
all four arms. Arbitrary caller detail moves through `nl13-reframe` as
structured data instead of pretending this seam is a general segmenter.

`cognition/tests/nl-lenses-13-live.fk` exposes all thirteen input phrases,
detections, next-lens outputs, generated frames, provenance stamps, and source
paths as live runtime data. The TypeScript live readout rendered all thirteen
human-readable rows, including `Я здесь. → 我在。`, `我在。 → 私はいます`,
`أنا. → नमस्ते।`, and `नमस्ते। → Buradayım.`; these arrows witness the bounded
phrase mapping only, not translation quality or semantic equivalence beyond
the seated presence meaning.

The remaining north star is learned, held-out language modeling and detection
with native-speaker-reviewed corpora. This organ is a truthful bridge to that
work, not its counterfeit completion.
