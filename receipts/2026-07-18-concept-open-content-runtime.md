# Arbitrary content-derived concept runtime

Date: 2026-07-18
Verdict: **one arbitrary address joined live; full multimodal goal remains open**

`presence/concept-open-content-runtime.fk` replaces the old integrated door's
tone and pixel-address acceptance with independently derived content evidence:

- generated text surface → complete Form 10,000-concept scan;
- speech-only PCM → pinned Whisper transcript → complete Form scan;
- decoded four-frame FFV1 video → Apple Vision OCR / pretrained labels →
  complete Form scan;
- generated PL source → exact recovery and an optional real language carrier;
- accepted text, audio, and video views → the existing persistent world model.

The request remains control-plane data. It can score an independently observed
candidate after inference, but it never creates a candidate or world entity.

## What this proves

- An arbitrary valid `(concept id, NL lens, PL lens)` can now invoke the same
  callable integration path; the courthouse fixture is no longer the only
  integrated address.
- Semantic acceptance does not use the older tone envelope or pixel identity
  band.
- Text, acoustic content, decoded-video content, and executable generated code
  can meet at one identity and persist as three independently admitted views.
- Misses in any modality remain visible and prevent `content-unified=1`.

## What this does not prove

- The speech is synthesized rather than held-out human speech.
- Generated video meaning is visible localized/English text plus host
  pretrained labels; it is not photorealistic scene generation.
- Whisper, Apple Vision, CoreText, ffmpeg, and the PL carrier remain host-rented.
- One joined row, twenty arbitrary audio probes, thirteen situated utterances,
  thirteen generated videos, and the real-photo corpora do not equal semantic
  observation of all 130,000 concept/lens addresses.
- Exact generated-source recovery is not arbitrary-source program
  understanding.

## Movement

What kept this build alive was making the control plane unable to masquerade as
perception. The most surprising teaching was that a narrow generated video can
be semantically honest when OCR is named as OCR, while still being far from a
learned visual world generator. Discomfort became gold when the attractive
single `unified=1` had to sit beside the much larger unobserved cross-product.
